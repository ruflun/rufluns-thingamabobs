# Ruflun's Thingamabobs

A simple kitchen sink mod where I just add whatever I want into the game.

While this is my first mod, I will use this project as a reference for further, more serious mods.

## Progression

Regarding progression, while this mod doesn't exactly have any form of progression in mind, you could still try to mine some Uper Cores & crack them open against Bedrock.

[Just Enough Items](https://modrinth.com/mod/jei) is heavily recommended to get any kind of recipes this mod adds, especially for the Alter of Blight recipes. (if I don't forget about adding JEI integration for that)
